Os Principais Comandos para Criar um Site

COMANDOS HTML

Alguns comandos utilizados para fazer o site

  • < html >é a base de qualquer página web. Ela envolve todo o conteúdo da sua página, desde o cabeçalho até o corpo. Sem ela, o navegador não saberia onde começa e termina o seu documento.
  • < img > é usada para adicionar imagens à sua página.
  • o termo h1,h2 o h seria para classificar como titulos e o numero que vem depois seria o que falaria qual titulo seria.
  • < p> é usado para colocar suas escritas na paginas.

    • COMANDOS CSS

    Comandos CSS utilizados no site

  • font-size serve para colocar o tamanho que irá ficar as coisa ou em pixels, em porcentagem entre outros
  • font-family serve para definir as fontes da letras
  • color serve para colocar cor nas letras
  • background-color serve para colocar fundo na sua paginas
  • width serve para alterar o tamanho das fotos que vc colocar no seu site, pode ser em pixels, porcentagem ou em outros tipos
