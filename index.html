Os Principais Comandos para Criar um Site

Comando HTML: <html>

A tag <html> é a raiz de um documento HTML. Todos os outros elementos do HTML devem estar dentro desta tag.

  • define o início e o fim do documento HTML.

    • Comando CSS: font-size

    A propriedade CSS font-size define o tamanho da fonte dos elementos. Pode ser especificado em pixels (px), ems, rems, etc.

  • É importante para garantir que o texto tenha o tamanho apropriado e seja legível.
